🧬 Formatos Biológicos

🧬 FASTA

Um arquivo FASTA é um formato de texto simples usado para armazenar sequências biológicas, como DNA, RNA ou proteínas.
Cada sequência é precedida por uma linha de identificação iniciada com o símbolo >, seguida por uma ou mais linhas contendo apenas a sequência de caracteres (A, T, C, G ou letras de aminoácidos).
É amplamente utilizado para representar sequências de referência ou dados já processados.

⚗️ FASTQ

Um arquivo FASTQ é um formato de texto utilizado para armazenar sequências biológicas obtidas por sequenciamento (como em NGS), junto com as pontuações de qualidade associadas a cada base.
Cada entrada ocupa quatro linhas: identificador (@), sequência, separador (+) e a linha de valores de qualidade (em código ASCII).
É o formato padrão para dados brutos de sequenciamento.
