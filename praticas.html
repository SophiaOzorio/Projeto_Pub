Objetivo: Baixar os dados necessários para o projeto.

Os alunos deverão escolher o tema de estudo e buscar amostras em bases de dados de exoma.

Alguns bancos de dados: NCBI, ENSEMBL, UCSC Genome Browser, TCGA e GEO. Links estão na aba "Referências".

Objetivo: Realizar o controle de qualidade das amostras baixadas.

Os alunos deverão utilizar a ferramenta FastQC para avaliar a qualidade das amostras.

Objetivo: Realizar a trimagem das amostras, se necessário.

Os alunos deverão remover sequências de baixa qualidade e adaptadores.

Objetivo: Realizar o alinhamento das amostras com o genoma de referência.

Objetivo: Seguir as boas práticas do GATK antes da chamada de variantes.

Objetivo: Realizar a chamada de variantes com HaplotypeCaller do GATK.

Objetivo: Realizar a anotação funcional das variantes.
