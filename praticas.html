Selecione uma prática abaixo para visualizar seu conteúdo.

Objetivo: Baixar os dados necessários para o projeto.

Nesta prática, os alunos deverão escolher o tema de estudo e buscar amostras em bases de dados para análise. Se atendem de selecionar dados de exoma!!

Alguns bancos de dados que podem ser utilizados: NCBI (National Center for Biotechnology Information), ENSEMBL, UCSC Genome Browser, TCGA (The Cancer Genome Atlas) e GEO (Gene Expression Omnibus). Links para os bancos se encontram na aba "Referências"
